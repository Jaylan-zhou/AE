Lab 1: Face Reconstruction with Autoencoder

1 Problem Statement
Participants will implement a Convolutional Autoencoder (AE) to reconstruct face images using
the CelebA dataset, which can be downloaded from Box. The objective is to understand latent
representation learning and image reconstruction.

2 Requirements
Participants should have access to a Python environment with PyTorch installed. Familiarity with
convolutional neural networks and loss functions is required.

3 Instructions

3.1 Task 1: Data Preparation and Visualization
• Use the Align&Cropped version of the CelebA dataset.
• Resize all images to 64 × 64 and normalize to [0, 1].
• Create training and validation dataloaders.
• Visualize one batch (e.g., 64 images in an 8 × 8 grid).

3.2 Task 2: Autoencoder Architecture Design
• Encoder:
  – 4 convolutional layers with output channels [32, 64, 128, 256]
  – Kernel size = 3, stride = 2, padding = 1
  – ReLU activation after each layer
  – Spatial resolution progression:
    64 → 32 → 16 → 8 → 4
  – Final feature map size: (256, 4, 4)
  – Flatten the feature map:
    256 × 4 × 4 = 4096
  – Use one fully connected layer to map:
    4096 → 100
    producing a 100-dimensional latent vector.

• Decoder:
  – Fully connected layer expanding latent vector:
    100 → 256 × 4 × 4
  – Reshape to (256, 4, 4)
  – 4 convolutional layers with bilinear upsampling
  – Spatial resolution progression:
    4 → 8 → 16 → 32 → 64
  – ReLU activation after each layer
  – Final Sigmoid layer to ensure output lies in [0, 1]

3.3 Task 3: Training
• Reconstruction loss (MSE):
  LMSE = (1/N) Σᵢ ||xᵢ − x̂ᵢ||²
• Optimizer: AdamW
• Train for 20–30 epochs
• Plot training and validation loss curves

3.4 Task 4: Evaluation
• Visualize input vs reconstructed images.
• Experiment with different latent dimensions (e.g., 32, 100, 256).

3.5 Task 5: Analysis
1. What information is stored in the latent space?
2. Why does the AE reconstruct images well but fail to reliably generate realistic new faces from
   random latent vectors?
3. How does latent dimension size affect reconstruction quality?
