## 5. 自编码器模型

- [x] 5.1 实现 `models/autoencoder.py`：Encoder 类
  - 4 层卷积网络（3→32→64→128→256 通道）
  - 每层：Conv2d(kernel=3, stride=2, padding=1) + ReLU
  - 展平并通过全连接层映射到潜在维度（4096 → latent_dim）
  - 实现 `forward` 方法返回潜在向量
- [x] 5.2 实现 `models/autoencoder.py`：Decoder 类
  - 全连接层扩展潜在维度（latent_dim → 4096）
  - 重塑为 (256, 4, 4) 特征图
  - 4 层转置卷积（256→128→64→32→3 通道）
  - 每层：ConvTranspose2d(kernel=4, stride=2, padding=1) + ReLU
  - 最后一层：Sigmoid 激活
  - 实现 `forward` 方法返回重建图像
- [x] 5.3 实现 `models/autoencoder.py`：Autoencoder 类
  - 组合 Encoder 和 Decoder
  - 实现 `forward` 方法：编码 → 解码
  - 提供获取编码器和解码器的方法
- [x] 5.4 实现模型权重初始化函数（Xavier 或 Kaiming 初始化）
- [ ] 5.5 测试模型前向传播：验证输入输出形状 (batch, 3, 64, 64)
