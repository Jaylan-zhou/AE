# 人脸重建自编码器实验提案

## 为什么

本项目旨在实现一个卷积自编码器（Convolutional Autoencoder）用于人脸图像重建，这是 DTS410TC 生成式人工智能课程的核心实验。通过使用 CelebA 数据集训练自编码器，我们能够深入理解潜在表示学习（latent representation learning）和图像重建的原理。完成此实验将为后续研究生成模型（如 VAE、GAN）奠定坚实基础。

## 变更内容

本实验将实现完整的自编码器训练和评估流程：

- **数据准备与可视化模块**
  - 下载并处理 CelebA Align&Cropped 数据集
  - 实现图像预处理：调整大小至 64×64，归一化到 [0, 1]
  - 创建训练集和验证集的 DataLoader
  - 实现批次图像可视化功能（8×8 网格显示 64 张图像）

- **自编码器模型架构**
  - 编码器（Encoder）：4 层卷积网络（32→64→128→256 通道）
  - 潜在空间：100 维向量
  - 解码器（Decoder）：4 层转置卷积 + 双线性上采样
  - 激活函数：ReLU + 输出层 Sigmoid

- **训练流程**
  - 损失函数：均方误差（MSE）
  - 优化器：AdamW
  - 训练 20-30 个 epoch
  - 实时记录和可视化训练/验证损失曲线

- **评估与分析工具**
  - 输入图像与重建图像的对比可视化
  - 不同潜在维度（32、100、256）的对比实验
  - 分析潜在空间存储的信息类型
  - 探讨自编码器在生成新样本方面的局限性

## 能力规格

### 新增能力

- `data-preparation`: 数据集加载、预处理和可视化功能
  - 处理 CelebA Align&Cropped 图像
  - 实现数据增强和归一化
  - 提供批次可视化工具

- `autoencoder-model`: 编码器-解码器架构实现
  - 实现指定的卷积层结构
  - 潜在向量压缩与重建
  - 支持可配置的潜在维度

- `training-loop`: 训练流程与损失跟踪
  - 实现完整的训练循环
  - 记录训练和验证损失
  - 生成损失曲线可视化

- `evaluation`: 模型评估与实验分析
  - 重建质量可视化
  - 不同潜在维度的对比实验
  - 潜在空间特性分析

### 修改的能力

无（这是全新项目，无需修改现有规格）

## 影响范围

- **代码结构**：创建完整的 PyTorch 项目结构，包括数据处理、模型定义、训练和评估模块
- **依赖项**：
  - Python 3.8+
  - PyTorch（用于深度学习框架）
  - torchvision（用于图像处理和数据增强）
  - matplotlib/seaborn（用于可视化）
  - numpy（用于数值计算）
  - PIL/Pillow（用于图像加载）
- **数据资源**：需要从 Box 下载 CelebA Align&Cropped 数据集
- **计算资源**：建议使用 GPU 加速训练（CPU 也可运行但速度较慢）
- **输出产物**：
  - 训练好的模型权重文件
  - 损失曲线图表
  - 重建图像对比图
  - 实验分析报告
